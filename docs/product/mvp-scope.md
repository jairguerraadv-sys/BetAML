# Escopo do MVP

MVP vendavel:

- Ingestao confiavel.
- Rules engine auditavel.
- Alertas.
- Case management.
- Report package.
- Audit logs.
- Dashboard operacional.
- Administracao de usuarios e API keys.
- Observabilidade basica.

Fora do MVP inicial:

- ML avancado obrigatorio.
- Feature store completa obrigatoria.
- Tuning automatico.
- Comunicacao regulatoria final automatica sem validacao juridica.

A plataforma auxilia a preparacao, governanca, rastreabilidade e evidencias para fluxos regulatorios, sem substituir validacao juridica ou responsabilidade final do operador.
