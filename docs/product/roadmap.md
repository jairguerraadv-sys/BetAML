# Roadmap

## Agora

- Hardening multi-tenant, RBAC e auth.
- Cadeia de custodia de report package.
- DLQ/replay e mapping versioning.
- CI/security gates.

## Proximo

- SSO/MFA.
- E2E cross-tenant.
- Nonce persistente para webhooks.
- Triggers DB para imutabilidade de audit/report.

## Depois

- Governanca ML completa com aprovacao formal.
- Drift monitoring automatizado.
- Integracoes regulatorias adicionais.
- Data quality scorecards por conector.
